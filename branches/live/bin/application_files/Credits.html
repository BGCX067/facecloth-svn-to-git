built with Processing
using the Video and traer.physics libraries

Key Assignments:
1 - lock/free the top-left corner point
2 - lock/free the top-right corner point
3 - lock/free the bottom-left corner point
4 - lock/free the bottom-right corner point
UP - increase gravity
DOWN - decrease gravity
SPACE - reset the environment
z - frame clear toggle
x - image screen toggle
c - corner point handle toggle
v - mesh outline toggle
b - grid toggle
